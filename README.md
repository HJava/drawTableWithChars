# drawTableWithChars

用纯字符画出表格。Draw tables with chars.

### 使用 Usage:

@param {Dom} 父级dom节点               
@param {Object} 描述表格内容对象

```javascript
drawTables.init(document.getElementById('ctn'), {
    title: ['name', 'age', 'sex'],
    direction: [0, 'age'],
    data: [
        ['haasdfha', 10, { content: 'heihei', col: 4, row: 3 }],
        ['hehe', 10],
        ['haasdfha', 10, 'male'],
        [
            { content: 'heihei', col: 4, row: 3 },
            { content: 'heihei', col: 4, row: 3 },
        ],
        [
            { content: 'heihei', col: 4, row: 3 },
            {}
        ]
    ]
});
```
